Python 基础

注释为 #

  • 数据类型和变量

    • 数据类型
      • 整数
      • 浮点数：科学计数法0.000012可以写成1.2e-5
      • 字符串：‘’单引号或“”双引号括起来，如果字符串包含单引号或双引号，则需要转义符\，如\’或\”，要是转义内容比较多，可以用r’ ‘
      • 布尔值：与或非运算为and, or, not。如 true or false，
      • 空值：none
      • 变量：可以是任意数据类型，可以多次赋值，可以多次赋不同类型的值，这种变量类型不固定的语言称为动态语言，c是静态语言
      • 常量：不能变的变量。在Python中，通常使用全部大写的变量名表示常量。其实常量也是可以变的[捂脸]，py没有任何机制拦着你去变化常量
      • 很多种除法：/是浮点数除，得到的永远是浮点数；//是地板除，得到的是向下取整的整数；%是取余除，得到的是余数
      • Bytes ：这是一种数据类型，形式为b’srting’或b”string”，每个字符占用一个字节

  • 字符串和编码

    • 字符编码：
      • Unicode：是不可变长编码，将所有语言都统一到一套编码里，全面杜绝乱码问题，带来新的问题，存储空间过大
      • UFT-8：是一种可变长编码，英文字母一个字节，汉字三个字节，生僻字符使用4-6个字节，兼容ASCII
        应用：记事本编辑，读取UTF-8码转换为Unicode码存到内存中，保存时再将Unicode码转为UTF-8保存到文件；浏览网页时服务器将Unicode码，转换为UTF-8传输到浏览器
    • Python的字符串
      • 最新的Python是以Unicode编码，一个字符对应多个字节
      • 对于单个字符，ord（）获取字符的整数编码表示，chr（）将编码转换为对应的字符
      • encode()和decode(): encode()将以Unicode表示的str转换为Bytes， decode（）反之
      • 格式化：即输出含有变量的字符串，如下，使用占位符%d，%f，%s，%x分别表示整数，浮点数，字符串，十六位整数，如果要输出%，则转义使用%%表示一个%

  • list和tuple

    • list：即列表，是一种有序的集合，同一列表中的元素类型可以不同，可以添加和删除其中的元素，形式为letter = [‘a’, ‘b’ , ‘c’]

      • 索引 ：类似C里的数组，如letter[0]是第一个元素，特点，索引可以为负，letter[-1]是最后一个元素
      • 添加元素： letter。append(‘d’)，追加元素；letter.insert(1, ‘e’)将‘e’插入索引为1的位置
      • 删除元素： letter.pop()，删除末尾的元素；letter.pop(0)，删除索引为1的元素

    • tuple：即元组，是另一种有序列表，一旦初始化就不能修改如果把列表比作变量，那元组就是常量，形式为number = (1, 2, 3)

      • 索引：与列表一样number[0]为第一个元素
      • append(),insert(),pop()？不存在的
      • 特殊情况：空的tuple number = （）； 一个元素的turple number = (1, ) ；turple元素可以是列表

  • 递归函数

    函数内部，可以调用其他函数，如果一个函数内部调用了自身，则这个函数是递归函数

    • 递归：优点是逻辑清晰，缺点是过深的调用会导致栈溢出

    • 尾递归：尾递归就是把当前的运算结果（或路径）放在参数里传给下层函数，可以对递归进行尾递归优化防止栈溢出

      个人理解是尾递归不需要保存本函数的运行状态，递归需要保存本函数的运行状态
