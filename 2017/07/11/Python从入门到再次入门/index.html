Python 基础

  • 数据类型和变量

    • 数据类型
      • 整数
      • 浮点数：科学计数法0.000012可以写成1.2e-5
      • 字符串：‘’单引号或“”双引号括起来，如果字符串包含单引号或双引号，则需要转义符\，如\’或\”，要是转义内容比较多，可以用r’ ‘

      • 空值：none
      • 变量：可以是任意数据类型，可以多次赋值，可以多次赋不同类型的值，这种变量类型不固定的语言称为动态语言，c是静态语言
      • 常量：不能变的变量。在Python中，通常使用全部大写的变量名表示常量。其实常量也是可以变的[捂脸]，py没有任何机制拦着你去变化常量
      • 很多种除法：/是浮点数除，得到的永远是浮点数；//是地板除，得到的是向下取整的整数；%是取余除，得到的是余数
      • Bytes ：这是一种数据类型，形式为b’srting’或b”string”，每个字符占用一个字节

      • 最新的Python是以Unicode编码，一个字符对应多个字节

  • list和tuple

    • list：即列表，是一种有序的集合，同一列表中的元素类型可以不同，可以添加和删除其中的元素，形式为letter = [‘a’, ‘b’ , ‘c’]

      • 索引 ：类似C里的数组，如letter[0]是第一个元素，特点，索引可以为负，letter[-1]是最后一个元素
      • 添加元素： letter。append(‘d’)，追加元素；letter.insert(1, ‘e’)将‘e’插入索引为1的位置
      • 删除元素： letter.pop()，删除末尾的元素；letter.pop(0)，删除索引为1的元素

    • tuple：即元组，是另一种有序列表，一旦初始化就不能修改如果把列表比作变量，那元组就是常量，形式为number = (1, 2, 3)

      • 索引：与列表一样number[0]为第一个元素
      • append(),insert(),pop()？不存在的
      • 特殊情况：空的tuple number = （）； 一个元素的turple number = (1, ) ；turple元素可以是列表

    • 尾递归：尾递归就是把当前的运算结果（或路径）放在参数里传给下层函数，可以对递归进行尾递归优化防止栈溢出

      个人理解是尾递归不需要保存本函数的运行状态，递归需要保存本函数的运行状态

模块

使用模块有什么好处？
最大的好处是大大提高了代码的可维护性。其次，编写代码不必从零开始。当一个模块编写完毕，就可以被其他地方引用。我们在编写程序的时候，也经常引用其他模块，包括Python内置的模块和来自第三方的模块。
使用模块还可以避免函数名和变量名冲突。相同名字的函数和变量完全可以分别存在不同的模块中，因此，我们自己在编写模块时，不必考虑名字会与其他模块冲突

  • 使用模块

    • Python的内置模块

      • 使用模块的第一步：使用import语句导入该模块，则有sys变量指向该模块，sys模块有一个argv变量，用list存储了命令行的所有参数。argv至少有一个元素，因为第一个参数永远是该.py文件的名称
    • 作用域
      • 公开变量（public）:正常的函数和变量名是公开的，如：abc,12d,Hi等
      • 特殊变量：类似“abc”,可以被直接引用，但留作特殊用途，一般不要使用这种变量名
      • 非公开变量（private）：类似“_abc”,这样的变量则不应该被直接引用，作为私有变量
        但是，Python并没有一种方法可以完全限制访问private函数或变量，这是一种好的编程习惯

  • 安装第三方模块

    • 在Python中，安装第三方模块是使用包管理工具pip完成的，如果操作系统是MAC或Linux，则无需安装pip，windows则需要安装pip，参考安装Python

面向对象编程

面向对象编程——Object Oriented Programming，简称OOP，是一种程序设计思想。OOP把对象作为程序的基本单元，一个对象包含了数据和操作数据的函数。面向对象的程序设计把计算机程序视为一组对象的集合，而每个对象都可以接收其他对象发过来的消息，并处理这些消息，计算机程序的执行就是一系列消息在各个对象之间传递。
数据封装、继承和多态是面向对象的三大特点

  • 类和实例

    面向对象最重要的概念就是类（class）和实例（instance）。类是抽象的模板；实例是根据类创建出来的一个个具体的“对象”，对象拥有相同的方法，但各自的数据不同。

    • 在Python中，通过class定义类，class后面紧接的就是类名，通常使用大写开头的单词，紧接的是小括号及其内容，表示该类的父类，如果没有合适的父类，则继承object类

    • 实例化，创建实例是通过类名加（）实现的，

    • 类的模板作用

      • 通过定义一个特殊的 __init__ 方法将一些必须绑定的属性写进去。注意__init__方法的第一个参数永远是self，表示创建的实例本身，可以将各种属性通过self绑定到实例本身

      • 有了__init__方法，在创建实例时就不能传入空的参数了，必须传入除了self之外的init相应的参数

  • 数据封装

    面向对象编程的一个重要特点就是数据封装。在上面的Student类中，每个实例就拥有各自的name和score这些数据。我们可以通过函数来访问这些数据，比如打印一个学生的成绩；但是，既然Student实例本身就拥有这些数据，要访问这些数据，就没有必要从外面的函数去访问，可以直接在Student类的内部定义访问数据的函数，这样，就把“数据”给封装起来了。这些封装数据的函数是和Student类本身是关联起来的，我们称之为类的方法

    • 定义方法：要定义一个方法，除了第一个参数是self外，其他和普通函数一样。要调用一个方法，只需要在实例变量上直接调用，除了self不用传递，其他参数正常传入

  • 访问限制

    如果要让内部属性不被外部访问，可以把属性的名称前加上两个下划线__，在Python中，实例的变量名如果以__开头，就变成了一个私有变量（private），只有内部可以访问，外部不能访问。

注意： 在Python中，变量名类似__xxx__的，也就是以双下划线开头，并且以双下划线结尾的，是特殊变量，特殊变量是可以直接访问的，不是private变量，所以，不能用__name__、__score__这样的变量名。有些时候，你会看到以一个下划线开头的实例变量名，比如_name，这样的实例变量外部是可以访问的，但是，按照约定俗成的规定，当你看到这样的变量时，意思就是，“虽然我可以被访问，但是，请把我视为私有变量，不要随意访问”。 Python本身没有任何机制阻止你干坏事，一切全靠自觉
实际上是Python解释器对外把name变量改成了_Studentname，所以，仍然可以通过_Studentname来访问name变量

  • 继承和多态

    • 继承

      在OOP程序设计中，当我们定义一个class的时候，可以从某个现有的class继承，新的class称为子类（Subclass），而被继承的class称为基类、父类或超类（Base class、Super class）

      • 比如，我们已经编写了一个名为Animal的class，有一个run()方法可以直接打印

      • 当我们需要编写Dog和Cat类时，就可以直接从Animal类继承

      • 通过继承，子类获得了父类的全部功能。由于Animial实现了run()方法，因此，Dog和Cat作为它的子类，什么事也没干，就自动拥有了run()方法

      • 也可以给子类增加一些方法

      当子类和父类都存在相同的run()方法时，我们说，子类的run()覆盖了父类的run()，在代码运行的时候，总是会调用子类的run()。这样，我们就获得了继承的另一个好处： 多态

    • 多态：多态是指一个函数在不同对象调用下会有不同的运行方式，可以通过子类重写父类的方法实现多态
      • 静态多态
      • 动态多态
      • 鸭子类型：动态语言的“鸭子类型”，它并不要求严格的继承体系，一个对象只要“看起来像鸭子，走起路来像鸭子”，那它就可以被看做是鸭子。
        Python的“feel-like object“就是一种鸭子类型。对真正的文件对象，它有一个read()方法，返回其内容。但是，许多对象，只要有read()方法，都被视为“feel-like object“。许多函数接收的参数就是“feel-like object“，你不一定要传入真正的文件对象，完全可以传入任何实现了read()方法的对象。

      并且还可以判断一个变量是否是某些类型中的一种
