  • xls to mat

    • %get the values in the Excel using xlsread.

    • %combine data as you want:

    • %save in mat file
    • save(matfileName,’AllData’);%In your matfile name

    • groups 查看当前登陆用户的组内成员

    • /etc/group文件包含所有组
    • /etc/shadow和/etc/passwd系统存在的所有用户名

  • 正确地复制ssh公钥到gitlab中

    • 避免用vim或gedit打开文件，用鼠标选中全部复制然后粘贴，貌似gitlab添加公钥时不会去除行之间的空格（github没有这个问题），而使用vim或gedit打开复制会连行间空格一起复制，结果就是添加的公钥不对
    • 正确的做法应该是使用xclip

  • 在一台电脑上使多个git帐号（github,gitlab等等）

    • 生成不同的密钥对(非必须，可以都使用同一个密钥)
      • ssh-keygen
      • ssh-add keyname

    • 编辑 ~/.ssh/config 文件

      • 配置如下,HostName是服务器的地址，User是用户名，PreferredAuthentications照抄即可，IdentityFile是使用的密钥对的私钥

    • 在网站中添加密钥对的私钥

      • 注意，复制公钥时要使用正确的方法：xclip

    • 测试ssh-key配置
